## Key Features

<AccordionGroup>
  <Accordion title="Smart Queue Mode" icon="list">
    Batch multiple prompts together and let AI agents process them in sequence. Perfect for planning multi-step tasks without interruption.
  </Accordion>

  <Accordion title="Autopilot Mode" icon="plane">
    Enable autonomous agent operation with customizable auto-responses. Let your agents work hands-free while maintaining control.
  </Accordion>

  <Accordion title="Ask User Tool" icon="circle-question">
    The core interaction primitive that enables human-in-the-loop workflows. Agents can request feedback, approval, or clarification at any point.
  </Accordion>

  <Accordion title="Session Management" icon="clock-rotate-left">
    Track tool call history, manage sessions, and maintain context across long-running workflows with built-in timeout and health warnings.
  </Accordion>

  <Accordion title="File References" icon="hashtag">
    Quick file and folder references with `#` autocomplete, plus context providers for terminal output and workspace diagnostics.
  </Accordion>

  <Accordion title="MCP Integration" icon="plug">
    Auto-registers with Kiro, Cursor, and Antigravity. Compatible with any MCP-enabled AI IDE for maximum flexibility.
  </Accordion>
</AccordionGroup>

## How It Works

<Steps>
  <Step title="Install TaskSync">
    Choose your integration method: VS Code extension, terminal protocol, or MCP server. Each option provides the same core functionality with different interfaces.
  </Step>
  
  <Step title="Configure Your Workflow">
    Set up Smart Queue Mode for batched responses, enable Autopilot for autonomous operation, or use direct interaction mode for real-time feedback.
  </Step>
  
  <Step title="Start Building">
    Your AI agents can now use the `ask_user` tool to request feedback, approval, or new tasks. Queue prompts in advance or let Autopilot handle responses automatically.
  </Step>
  
  <Step title="Track Progress">
    Monitor tool call history, manage session state, and review past interactions. TaskSync maintains full context across your workflow.
  </Step>
</Steps>
