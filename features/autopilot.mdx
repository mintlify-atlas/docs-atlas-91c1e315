**Autopilot Mode** automatically responds to `ask_user` tool calls without waiting for manual input. Configure custom responses to guide the agent's autonomous behavior.
</Info>

## Overview

<Card title="Autonomous Operation" icon="robot">
When Autopilot is enabled, TaskSync automatically responds to `ask_user` tool calls using your configured response text. This allows AI agents to continue working without human intervention.
</Card>

## How It Works

1. **Enable Autopilot**: Toggle the Autopilot switch in the TaskSync sidebar
2. **Configure Response**: Set custom response text or use cycling prompts
3. **Automatic Responses**: When the AI triggers `ask_user`, TaskSync sends the configured response
4. **Queue Priority**: Queued prompts always override Autopilot responses

<Warning>
Autopilot continues until manually disabled or the auto-response limit is reached (default: 5 consecutive responses).
</Warning>

## Autopilot Processing Logic

```typescript
// From webviewProvider.ts:867-929
if (this._autopilotEnabled && !(this._queueEnabled && this._promptQueue.length > 0)) {
    // Cancel any superseded pending request
    this._cancelSupersededPendingRequest();

    this._consecutiveAutoResponses++;

    const config = vscode.workspace.getConfiguration('tasksync');
    const maxConsecutive = config.get<number>('maxConsecutiveAutoResponses', 5);

    if (this._consecutiveAutoResponses > maxConsecutive) {
        this._autopilotEnabled = false;
        await config.update('autopilot', false, vscode.ConfigurationTarget.Workspace);
        this._updateSettingsUI();
        vscode.window.showWarningMessage(
            `TaskSync: Auto-response limit (${maxConsecutive}) reached.`
        );
    } else {
        const toolCallId = `tc_${Date.now()}_${Math.random().toString(36).substring(2, 9)}`;
        this._currentToolCallId = toolCallId;

        // Apply human-like delay (jitter)
        await this._applyHumanLikeDelay('Autopilot');

        // Re-check after delay: user may have disabled autopilot
        if (!this._autopilotEnabled || this._currentToolCallId !== toolCallId) {
            // State changed during delay — fall through to normal flow
        } else {
            // Get cycling prompt or fallback text
            let effectiveText: string;
            if (this._autopilotPrompts.length > 0) {
                effectiveText = this._autopilotPrompts[this._autopilotIndex];
                this._autopilotIndex = (this._autopilotIndex + 1) % this._autopilotPrompts.length;
            } else {
                effectiveText = this._normalizeAutopilotText(this._autopilotText);
            }
            
            vscode.window.showInformationMessage(
                `TaskSync: Autopilot auto-responded. (${this._consecutiveAutoResponses}/${maxConsecutive})`
            );

            const entry: ToolCallEntry = {
                id: toolCallId,
                prompt: question,
                response: effectiveText,
                timestamp: Date.now(),
                isFromQueue: false,
                status: 'completed'
            };
            this._currentSessionCalls.unshift(entry);
            this._currentSessionCallsMap.set(entry.id, entry);
            this._updateViewTitle();
            this._updateCurrentSessionUI();
            this._currentToolCallId = null;

            return {
                value: effectiveText,
                queue: this._queueEnabled && this._promptQueue.length > 0,
                attachments: []
            };
        }
    }
}
```

## Human-like Delays

<Card title="Natural Timing" icon="clock">
Autopilot responses include optional random delays (2-6s) to simulate human reading and response time.
</Card>

From `webviewProvider.ts:428-456`:

```typescript
/**
 * Generate a random delay (jitter) between min and max seconds.
 * 
 * Random delays simulate natural human pacing — the time it takes
 * to read a message and type a response. Fixed delays create
 * unnatural patterns; jitter produces realistic timing variation.
 */
private _getHumanLikeDelayMs(): number {
    if (!this._humanLikeDelayEnabled) {
        return 0;
    }
    const minMs = this._humanLikeDelayMin * 1000;
    const maxMs = this._humanLikeDelayMax * 1000;
    // Uniform random distribution across the range
    return Math.floor(Math.random() * (maxMs - minMs + 1)) + minMs;
}

/**
 * Wait a random duration before sending an automated response.
 * Called before Autopilot, Queue, and Timeout auto-responses.
 */
private async _applyHumanLikeDelay(label?: string): Promise<void> {
    const delayMs = this._getHumanLikeDelayMs();
    if (delayMs > 0) {
        const delaySec = (delayMs / 1000).toFixed(1);
        if (label) {
            vscode.window.setStatusBarMessage(
                `TaskSync: ${label} responding in ${delaySec}s...`, 
                delayMs
            );
        }
        await new Promise(resolve => setTimeout(resolve, delayMs));
    }
}
```

## Backward Compatibility

<Tip>
Autopilot automatically migrates from the old `autoAnswer` settings for backward compatibility.
</Tip>

From `webviewProvider.ts:595-637`:

```typescript
private _loadSettings(): void {
    const config = vscode.workspace.getConfiguration('tasksync');
    
    // Backward-compatible migration: read old 'autoAnswer'/'autoAnswerText' keys
    // if new 'autopilot'/'autopilotText' keys haven't been set by user
    const inspectedAutopilot = config.inspect<boolean>('autopilot');
    const hasNewAutopilotKey = inspectedAutopilot?.globalValue !== undefined
        || inspectedAutopilot?.workspaceValue !== undefined
        || inspectedAutopilot?.workspaceFolderValue !== undefined;

    if (!hasNewAutopilotKey) {
        const oldVal = config.inspect<boolean>('autoAnswer');
        const hasOldKey = oldVal?.globalValue !== undefined
            || oldVal?.workspaceValue !== undefined
            || oldVal?.workspaceFolderValue !== undefined;
        if (hasOldKey) {
            this._autopilotEnabled = config.get<boolean>('autoAnswer', false);
        } else {
            this._autopilotEnabled = false;
        }
    } else {
        this._autopilotEnabled = config.get<boolean>('autopilot', false);
    }
    // ... similar migration for autopilotText
}
```
