<Info>
**Smart Queue Mode** allows you to queue multiple prompts ahead of time. When the AI agent triggers the `ask_user` tool, TaskSync automatically sends the next queued prompt instead of waiting for manual input.
</Info>

## Overview

<CardGroup cols={2}>
  <Card title="Normal Mode" icon="message">
    Each `ask_user` call waits for your manual response. You type and send each answer individually.
  </Card>
  <Card title="Queue Mode" icon="layer-group">
    Pre-queue multiple prompts. TaskSync auto-responds to `ask_user` calls by sending queued prompts in order.
  </Card>
</CardGroup>

## How It Works

1. **Toggle Queue Mode**: Click the queue toggle button in the TaskSync sidebar
2. **Add Prompts**: Type prompts and click "Add to Queue" instead of sending immediately
3. **Auto-Response**: When the AI triggers `ask_user`, TaskSync automatically sends the next queued prompt
4. **Human-like Delays**: Optional random delays (2-6s) simulate natural response timing

<Tip>
Queued prompts **always take priority** over Autopilot. If both are enabled, queue prompts are sent first.
</Tip>

## Queue State Persistence

<Note>
Queue state is **persisted per workspace** and survives VS Code restarts. Queue data is stored in workspace-scoped storage and saved with a 300ms debounce.
</Note>

```typescript
// From webviewProvider.ts:159
private readonly _QUEUE_SAVE_DEBOUNCE_MS = 300;
```

### Storage Location

Queued prompts are saved to:
```
~/.config/Code/User/workspaceStorage/<workspace-id>/tasksync-queue.json
```

## Queue Processing Logic

```typescript
// From webviewProvider.ts:958-996
if (this._queueEnabled && this._promptQueue.length > 0) {
    const queuedPrompt = this._promptQueue.shift();
    if (queuedPrompt) {
        this._saveQueueToDisk();
        this._updateQueueUI();

        // Random delay simulates human reading/response time
        await this._applyHumanLikeDelay('Queue');

        // Re-check after delay: user may have disabled queue
        if (!this._queueEnabled || this._currentToolCallId !== toolCallId) {
            // State changed — restore prompt to queue
            this._promptQueue.unshift(queuedPrompt);
            this._saveQueueToDisk();
            this._updateQueueUI();
        } else {
            // Create completed tool call entry for queue response
            const entry: ToolCallEntry = {
                id: toolCallId,
                prompt: question,
                response: queuedPrompt.prompt,
                timestamp: Date.now(),
                isFromQueue: true,
                status: 'completed'
            };
            this._currentSessionCalls.unshift(entry);
            this._currentSessionCallsMap.set(entry.id, entry);
            
            return {
                value: queuedPrompt.prompt,
                queue: this._queueEnabled && this._promptQueue.length > 0,
                attachments: queuedPrompt.attachments || []
            };
        }
    }
}
```

## Human-like Delays

<Card title="Natural Timing" icon="clock">
Optional random delays (2-6s by default) simulate human reading and typing time for more realistic agent workflows.
</Card>

### Configuration

```json
{
  "tasksync.humanLikeDelay": true,
  "tasksync.humanLikeDelayMin": 2,
  "tasksync.humanLikeDelayMax": 6
}
```

From `package.json:188-209`:

